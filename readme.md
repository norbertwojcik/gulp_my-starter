# My Gulp starter for projects #

## 1.Global installation ##
* npm install -g gulp

## 2. Install npm ##
* npm install 

## 3. Install npm plugins ##
* npm install --save-dev ........ / npm i -D .......

## 4. Build project ##
* gulp build

## 5. Watch file ##
* gulp

## Update modules ##
1) npm install -g npm-check-updates
2) npm-check-updates -u
3) rm -fr node_modules
4) npm install

or

1) npm-check-updates -u
2) npm update
